  • 基本

  • Lynx-Chan是什么?

    Lynx-Chan是中文匿名讨论社区.
    我们称Lynx-Chan为野猫(猫岛).
    本站无论前端后端均采用匿名,不收集用户私人信息(IP地址无法避免,但你可以使用代理或Tor隐藏真实IP)，
    本站类似英文4chan,繁体Komica,简体A岛(非真正匿名).
    本站和类似中文网站无任何关系,更不存在任何利益.

  • 需要注册吗?

    任何人发表帖子\评论(回复)\分享图像或视频\文件都不需要注册(创建版块除外).

  • 为什么有的板块需要输入验证码?

    为了防止垃圾帖子.

  • 如何发主题贴(线程)?

    首先选择正确的版块(例如:b板)，
    昵称、主题可留空,
    在【评论】内输入你要发的内容，
    并输入【验证码】，
    最后点【发贴】。
    好了，就这么简单。
    ps：其它选项可省略不写，默认-匿名,位置-启用。

  • 如何回复(回贴)?

    有两种回复方式
    1,简单粗暴，直接点【回复】。
    2,快速回复，点#7后面的数字7。

  • 如何隐藏线程？

    点击帖子左侧“-”号.

  • 无法发贴?

    出现“密密太长X个字符以内”提示，清空主题框、邮箱、密码框内内容。
    ps:这是浏览器自动填充内容引起的错误。

  • 版块

  • 如何创建新版块?

    首先必须注册帐户(左上方或侧边栏低部"登录")，
    登录你的帐户，找到【创建电路板】，
    输入相关内容(电路板URL必须为字母)，
    最后点【创建电路板】。
    好了，就这么简单。

  • 如何删除自己在贴图板发的帖子或图片?

    单帖:点击每个帖子右端"!"号可删除单帖。
    多帖:先选择多个帖子后在页面底部找到并点击【删贴/举报】

  • 其它

  • 如何举报?

    点击帖子左侧“!”号.

  • 如何使用设置功能添加CSS背景图？

    在顶部》【设置】》【CSS】中添加css样式代码可改变版块风格
    例：添加下方css代码为版块增加背景图

    body { background: #fff 0 20px url('https://dss3.bdstatic.com/iPoZeXSm1A5BphGlnYG/skin/822.jpg?2') no-repeat; left top; }
    ps:你可以任意添加修改CSS。
